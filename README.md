9ª Festa da Pizza - ETEC de Fernandópolis

Este projeto é um site desenvolvido para a divulgação da 9ª Festa da Pizza da ETEC de Fernandópolis. O objetivo é informar a comunidade sobre o evento e facilitar o acesso a detalhes importantes.

Tecnologias e Infraestrutura

  - Frontend: HTML5, CSS3 e JavaScript.
  - Hospedagem: Google Firebase Hosting.
  - Automação (CI/CD): GitHub Actions integrado ao Firebase para deploy automático.
